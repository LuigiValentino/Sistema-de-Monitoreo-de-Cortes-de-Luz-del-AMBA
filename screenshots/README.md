# Capturas de Pantalla

Este directorio debe contener las capturas de pantalla del proyecto.

## Capturas necesarias:

1. **dashboard.png** - Vista principal del dashboard
2. **mapa.png** - Mapa interactivo con los cortes
3. **informacion.png** - Pestaña de información con la tabla
4. **acerca.png** - Pestaña Acerca de

## Cómo tomar las capturas:

1. Ejecutar la aplicación: `python app.py`
2. Abrir http://localhost:5000 en el navegador
3. Usar herramienta de captura de pantalla o Print Screen
4. Guardar las imágenes en esta carpeta con los nombres indicados

## Notas:
- Las imágenes deben ser formato PNG
- Se recomienda un tamaño de alrededor de 1200x800 px
- Asegurate de mostrar el contenido completo de cada sección
