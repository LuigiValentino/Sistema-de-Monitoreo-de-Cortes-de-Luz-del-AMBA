Sistema de Monitoreo de Cortes de Luz del AMBA

Seguimiento en tiempo real de los cortes de energía eléctrica en el Área Metropolitana de Buenos Aires

Versión 1.0.0 | Actualizado: -

Acerca de la Aplicación

Monitor de Cortes de Luz es una aplicación de seguimiento en tiempo real de los cortes de energía eléctrica en el Área Metropolitana de Buenos Aires (AMBA). Recopila datos oficiales del ENRE (Ente Nacional de Regulación de la Electricidad) sobre interrupciones de servicio de EDESUR y EDENOR, actualizándose automáticamente cada 5 minutos.

  • Datos: Información oficial del ENRE
  • Cobertura: 20 partidos del AMBA
  • Tipo de cortes: Media Tensión (MT) y Baja Tensión (BT)
  • Empresas: EDESUR (sur) y EDENOR (norte)

Cómo Usar

Esta herramienta te permite monitorear los cortes de luz en el AMBA de manera sencilla. Desde el Dashboard podés ver estadísticas generales, el mapa interactivo con la ubicación de los cortes y una lista de los cortes activos recientes. Podés filtrar por empresa (EDESUR/EDENOR) y tipo de tensión usando los menús desplegables.

  • Dashboard: Vista principal con estadísticas y mapa
  • Mapa: Clic en círculos para ver detalles
  • Gráficos: Evolución histórica 7/14/30 días
  • Información: Tabla completa con búsqueda

Cómo Funciona el Monitoreo

El sistema funciona mediante un monitoreo automático que se ejecuta en segundo plano en el servidor local. Un script de Python extrae los datos publicados por el ENRE y el scheduler (APScheduler) ejecuta la actualización cada 5 minutos automáticamente. Los datos se almacenan en una base de datos SQLite local con historial, las direcciones se geocodifican para mostrarse en el mapa, y Flask serve los datos al frontend mediante una API REST.
